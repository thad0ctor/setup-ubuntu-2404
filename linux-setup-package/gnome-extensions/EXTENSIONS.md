# GNOME Shell Extensions

These extensions were detected on your original Ubuntu installation.

## How to Install

### Method 1: Using Extension Manager (Recommended)
1. Open "Extension Manager" from your applications menu
2. Search for each extension by name
3. Click "Install" and then enable them

### Method 2: From Website
Visit https://extensions.gnome.org/ and search for each extension

## Extensions List

### Dash/Panel Extensions
- **Dash to Dock** (dash-to-dock@micxgx.gmail.com)
  - Adds a macOS-like dock to your desktop
  - Highly customizable positioning and appearance

- **Dash to Panel** (dash-to-panel@jderose9.github.com)
  - Combines top panel with dash for a Windows-like taskbar
  - Note: May conflict with Dash to Dock - choose one

### System Indicators
- **App Indicator Support** (appindicatorsupport@rgcjonas.gmail.com)
  - Adds support for application tray icons
  - Required for many applications to show system tray icons

- **Workspace Indicator** (workspace-indicator@gnome-shell-extensions.gcampax.github.com)
  - Shows current workspace in the panel
  - Built-in GNOME extension

### Utilities
- **Show Desktop Applet** (show-desktop-applet@valent-in)
  - Adds a show desktop button to the panel

- **Places Menu** (places-menu@gnome-shell-extensions.gcampax.github.com)
  - Quick access to folders and locations
  - Built-in GNOME extension

- **Extension List** (extension-list@tu.berry)
  - Shows list of installed extensions in the panel

- **Tiling Assistant** (tiling-assistant@ubuntu.com)
  - Enhanced window tiling features
  - Ubuntu default extension

### Recording/Streaming
- **EasyScreenCast** (EasyScreenCast@iacopodeenosee.gmail.com)
  - Screen recording tool
  - Simple interface for capturing screen

### Desktop
- **Desktop Icons NG (DING)** (ding@rastersoft.com)
  - Shows desktop icons
  - Ubuntu default extension

### Other
- **Ubuntu AppIndicators** (ubuntu-appindicators@ubuntu.com)
  - Ubuntu-specific app indicator support

- **Ubuntu Dock** (ubuntu-dock@ubuntu.com)
  - Ubuntu's default dock

- **Auto Move Windows** (auto-move-windows@gnome-shell-extensions.gcampax.github.com)
  - Automatically move applications to specific workspaces

- **Docker Extension** (docker@stickman_0x00.com)
  - Docker container management (if you use Docker)

## Recommended Configuration

### For a Windows-like Experience:
- Dash to Panel
- App Indicator Support
- Tiling Assistant
- Show Desktop Applet

### For a macOS-like Experience:
- Dash to Dock
- App Indicator Support
- Tiling Assistant

### Minimal Setup:
- App Indicator Support (essential for tray icons)
- Tiling Assistant (better window management)
- Desktop Icons NG (if you use desktop icons)

## Notes

- Some extensions may not be compatible with the latest GNOME version
- After installing, you may need to log out and log back in
- Extensions can be managed in Extension Manager or GNOME Tweaks
- You can enable/disable extensions without uninstalling them

## Current GNOME Version

Check your GNOME version with:
```bash
gnome-shell --version
```

Make sure extensions are compatible with your GNOME version.
